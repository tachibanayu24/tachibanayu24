## 要約

2018年に新卒でNTTコムウェア株式会社に企画職で入社し、主に官公庁に向けドローンやXR、自動運転領域の先端技術応用提案などに従事しました。  
その後ソフトウェアエンジニアとして複数組織で様々な領域のサービス開発に取り組みました。組織開発や採用、育成、技術戦略立案、品質管理、PM、PdM、開発責任者など幅広く取り組みました。
