# tachibanayu24.com 仕様書

## 概要

tachibanayu24 の名刺とかに QR コードとして入れたりするリンク集です。

---

## 機能

### プロフィール表示

- 名前、自己紹介、所属企業を表示
- 各種 SNS・外部サービスへのリンクをアイコン付きで一覧表示

### 多言語対応

- 日本語 / 英語の切り替えが可能
- 選択言語はブラウザに保存され、次回以降も維持される

### 時間帯連動の背景演出

訪問者のローカル時刻に応じて背景が変化する。

| 時間帯 | 時間        | 雰囲気                                 |
| ------ | ----------- | -------------------------------------- |
| 朝     | 5:00-11:00  | 柔らかい朝日と清潔なベッドシーツ、朝靄 |
| 昼     | 11:00-17:00 | 澄み渡る青空と太陽の光                 |
| 夕     | 17:00-20:00 | 夕日と長く伸びる影                     |
| 夜     | 20:00-5:00  | 夜空と雲、蛍の光                       |

- 時間帯の境界では 30 分かけてなめらかに遷移
- UI の配色も時間帯に合わせて変化

---

## コンテンツ管理

プロフィール情報は `me.json` で一元管理。更新すると GitHub の README も自動再生成される。

---

## 開発者向け

- `#debug` を URL に付与するとデバッグパネルが表示され、時間帯を手動切替可能

## 今後の実装予定

### 1. インタラクティブなうさぎキャラクター 🐰

#### 概要

背景にうさぎのキャラクターを配置し、時間帯に応じて動作が変わる。訪問者がクリックすると反応する。

#### 仕様

**配置**

- 画面右下に小さめのシルエットとして配置
- 背景レイヤー（Canvas）に描画
- カードUIの邪魔にならない位置

**時間帯別の動作**

| 時間帯       | 動作         | 詳細                             |
| ------------ | ------------ | -------------------------------- |
| 朝 (MORNING) | 元気に跳ねる | 小さくぴょんぴょん、耳がぴくぴく |
| 昼 (NOON)    | 草を食べてる | もぐもぐアニメーション           |
| 夕 (EVENING) | のんびり座る | たまに耳が動く程度               |
| 夜 (NIGHT)   | 眠っている   | ZZZ...、呼吸で軽く上下           |

**インタラクション**

- クリック → ぴょんと跳ねる + ハートが出る（全時間帯共通）
- 夜にクリック → 一瞬起きて、また眠る
- ホバー → 耳がぴくっと反応（デスクトップのみ）

**デザイン**

- シンプルなシルエット or ミニマルなイラスト
- 背景に溶け込む色合い（時間帯に応じて変化）
- サイズ: 50-80px 程度

#### 技術的な実装方針

- `background/effects/` に新しいエフェクトとして追加
- `BaseEffect` を継承した `RabbitCharacter` クラス
- 時間帯に応じた `state`: `jumping`, `eating`, `sitting`, `sleeping`
- クリックイベントは Canvas の座標計算で判定
- スプライトアニメーション or パス描画

#### ファイル構成

```
background/
├── effects/
│   ├── rabbit-character.js  # 新規作成
│   └── ...
├── assets/
│   └── rabbit-sprites.png   # スプライト画像（オプション）
└── config.js                # RABBIT 設定追加
```

### 2. 名刺の「裏面」コンセプト

#### 概要

カードをクリック/タップするとフリップして裏面が表示される。物理的な名刺では不可能なインタラクティブ体験を提供。

#### 仕様

**トリガー**

- カード右上に小さなフリップアイコン/ボタン
- またはカード全体をクリック（リンク以外の部分）

**表面（現在の内容）**

- アバター
- 名前
- 自己紹介（短め）
- 所属企業
- SNSリンク

**裏面（新規追加）**

```
┌─────────────────────────────────────┐
│                     [↻ Flip]        │
│                                     │
│  💼 What I Do                       │
│  ─────────────────────────────────  │
│  スタートアップの0→1開発が得意です。   │
│  フロントエンド/バックエンド両方      │
│  対応できます。                      │
│                                     │
│  🛠 Tech Stack                      │
│  ─────────────────────────────────  │
│  [TypeScript] [React] [Node.js]    │
│  [Go] [AWS] [GCP]                  │
│                                     │
│  🐰 About Me                        │
│  ─────────────────────────────────  │
│  • うさぎ（名前: ○○）と暮らしてます  │
│  • 週5ジム、週3サウナ               │
│  • 低温調理にハマり中                │
│                                     │
│  [☕ 話しませんか？]                 │
│  └→ Calendly or メールリンク        │
│                                     │
└─────────────────────────────────────┘
```

**アニメーション**

- 3D フリップ（Y軸回転）
- 回転時間: 0.6s 程度
- イージング: ease-in-out

**レスポンシブ**

- モバイル: スワイプでもフリップ可能（オプション）
- 裏面のコンテンツはスクロール可能（内容が多い場合）

#### 技術的な実装方針

- CSS `transform-style: preserve-3d` + `perspective`
- `.card` を `.card-inner`, `.card-front`, `.card-back` に分割
- JavaScript は状態（表/裏）のトグルのみ
- 裏面のコンテンツも `me.json` で管理

#### CSS構造

```css
.card-container {
  perspective: 1000px;
}

.card-inner {
  transform-style: preserve-3d;
  transition: transform 0.6s ease-in-out;
}

.card-inner.flipped {
  transform: rotateY(180deg);
}

.card-front,
.card-back {
  backface-visibility: hidden;
}

.card-back {
  transform: rotateY(180deg);
}
```

## 将来の検討事項（保留）

### アクティビティフィード

GitHub、Zenn、Note、Qiita などから最新のアクティビティを自動取得して表示。

**取得可能なデータ:**

| サービス | 方法       | 内容                    |
| -------- | ---------- | ----------------------- |
| GitHub   | Public API | スター、Push、PR、Issue |
| Zenn     | RSS        | 記事タイトル、日付      |
| Note     | RSS        | 記事タイトル、日付      |
| Qiita    | API        | 記事タイトル、いいね数  |

**技術的課題:**

- RSS は CORS 制限あり → Cloudflare Workers でプロキシ必要
- または GitHub Actions でビルド時に静的生成
